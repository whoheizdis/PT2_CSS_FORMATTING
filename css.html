Introduction to CSS

CSS stands for Cascading Style Sheets, a language used to style and design web pages. It controls how HTML elements are displayed on the screen, such as colors, fonts, spacing, and layout.

CSS works together with HTML and JavaScript:

  • HTML provides the structure of the webpage
  • CSS controls the appearance and design
  • JavaScript adds interactivity and behavior

By separating design from content, CSS allows developers to easily manage and update the look of multiple pages at once.

Why CSS is Used

CSS is widely used in web development because it helps create visually appealing and well-organized websites.

Some reasons for using CSS include:

  • Better design and layout of webpages
  • Consistent styling across multiple pages
  • Faster website maintenance since styles can be changed in one file
  • Improved performance and user experience
  • Responsive design that adapts to different screen sizes and devices

How CSS Works

CSS applies styles to HTML elements using rules and selectors. A selector identifies which HTML element should receive the style.

When a webpage loads:

  • The browser loads and reads the HTML file.
  • The browser loads any CSS files linked to the page.
  • The browser matches CSS rules to HTML elements.
  • The final styles are applied and displayed on the screen.

This process allows the browser to turn a plain webpage into a visually styled layout.

CSS Syntax and Rules

CSS is written using rulesets, which consist of a selector and declarations.

Parts of CSS syntax:

  • Selector – the HTML element being styled
  • Property – the style being changed (color, font, size, etc.)
  • Value – the setting applied to the property

Ways to Add CSS to HTML

CSS can be added to HTML in three main ways:

Inline CSS - Applied directly inside an HTML element using the style attribute.


Internal CSS - Placed inside a [style] tag within the [head] section of an HTML document.


External CSS - Stored in a separate .css file and linked to the HTML file. External CSS is the most common method because it allows one stylesheet to control multiple pages.

CSS Selectors

Selectors are patterns used to choose which HTML elements will receive specific styles.

Selector Description
Element selector selects elements by tag name
Class selector selects elements with a specific class
ID selector selects an element with a unique ID
Attribute selectors target elements with specific attributes
Pseudo-classes and pseudo-elements style elements in special states or parts (like :hover or ::first-letter)

The Cascading Concept

The word “cascading” refers to how CSS decides which style rule should apply when multiple rules target the same element.

Priority depends on:

  • Inline styles
  • Internal styles
  • External styles
  • Selector specificity
  • Source order (later rules can override earlier ones)

This system ensures the correct styles are applied to each element.

Advantages of CSS

CSS provides several advantages in web development:

  • Separates content from design
  • Makes websites easier to maintain
  • Reduces repeated code
  • Allows responsive layouts for different devices
  • Improves performance and readability of web pages
