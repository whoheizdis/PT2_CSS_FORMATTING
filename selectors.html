Introduction to CSS Selectors

CSS selectors are patterns used to select and target HTML elements so styles can be applied to them. They tell the browser which elements on a webpage should receive specific styling rules such as colors, fonts, spacing, and layout.

Selectors help developers apply styles efficiently and consistently across a webpage. Without selectors, CSS would not know which elements should be styled.

CSS selectors can generally be grouped into five categories:

  • Simple selectors
  • Combinator selectors
  • Pseudo-class selectors
  • Pseudo-element selectors
  • Attribute selectors

Simple Selectors

Simple selectors select elements based on basic information like tag name, class, or ID.


Element Selector

The element selector targets HTML elements by their tag name.

This example applies the style to all [p] elements on the page.


ID Selector

The ID selector selects a specific element using its unique id attribute.

Since IDs are unique, this selector usually targets one element only.


Class Selector

The class selector selects elements with a specific class attribute.

Multiple elements can share the same class, allowing them to have the same style.


Universal Selector

The universal selector (*) selects all elements on a webpage.

This applies the style to every element on the page.


Grouping Selector

The grouping selector applies the same styles to multiple elements by separating selectors with commas.

This reduces repeated code in CSS.

Combinator Selectors

Combinator selectors define relationships between elements in the HTML structure.


Descendant Selector (space)

Targets elements inside another element.

This selects all [p] elements inside a [div].


Child Selector (>)

Selects only direct children of a parent element.

Only [p] elements that are direct children of [div] are styled.


Adjacent Sibling Selector (+)

Selects an element that comes immediately after another element.


General Sibling Selector (~)

Selects elements that share the same parent and appear after a specified element.

Attribute Selectors

Attribute selectors target elements based on their attributes or attribute values.

This selector styles only [input] elements where the type attribute is "text".

Attribute selectors can also:

  • Select elements with a specific attribute
  • Select elements with a specific attribute value
  • Match attribute values that start, end, or contain certain text.

Pseudo-Class and Pseudo-Element Selectors

These selectors target elements based on state or specific parts of elements.


Pseudo-class

Pseudo-classes style elements in a specific state.

Examples:

  • :hover – when the mouse is over an element
  • :active – when an element is being clicked
  • :visited – links that have already been visited

Pseudo-Element

Pseudo-elements style specific parts of an element.

Examples:

  • ::first-letter – styles the first letter of text
  • ::before – inserts content before an element
  • ::after – inserts content after an element
