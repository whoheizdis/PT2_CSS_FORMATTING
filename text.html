Text formatting with css

Introduction to CSS Text Formatting

CSS text formatting refers to using CSS properties to control the appearance and layout of text on a webpage. These properties allow developers to modify color, alignment, spacing, decoration, and transformation of text.

Text formatting is important because a large portion of website content is text. Proper formatting improves readability, user experience, and visual design of web pages.

Common text formatting features include:

  • Changing text color
  • Aligning text
  • Adding spacing between letters and words
  • Decorating text with lines
  • Transforming text to uppercase or lowercase

Text Color

The color property is used to set the color of text in CSS.

The color can be specified in different formats such as:

  • Color names (e.g., red, blue)
  • HEX values (e.g., #ff0000)
  • RGB values (e.g., rgb(255,0,0))

Good contrast between text color and background color is important so the text is easy to read.

Text Alignment

The text-align property controls the horizontal alignment of text inside an element.

Value Description
left aligns text to the left
right aligns text to the right
center centers the text
justify spreads text evenly across the line

Text Decoration

The text-decoration property adds decorative lines to text.

Value Description
underline adds a line below text
overline adds a line above text
line-through adds a strike-through line
none removes decoration (often used for links)

Text Transformation

The text-transform property controls the capitalization of text.

Value Description
uppercase converts text to capital letters
lowercase converts text to small letters
capitalize capitalizes the first letter of each word

Text Spacing

CSS allows developers to control spacing between letters, words, and lines to improve readability.


Letter Spacing

The letter-spacing property adjusts the space between characters.


Word Spacing

The word-spacing property adjusts the space between words.


Line Height

The line-height property controls the space between lines of text.

Other Text Properties

CSS also provides additional text formatting properties such as:

Property Description
text-indent indents the first line of a paragraph
text-shadow adds shadow effects to text
direction defines text direction (left-to-right or right-to-left)
text-overflow controls how hidden text is displayed when it overflows a container

These properties allow developers to create visually appealing text styles and layouts.
